# Yuk Grill Yuk

Aplikasi ini dibuat untuk usaha kalian.

Semoga usaha kita terus berkembang, laris, dan berkah.

**Tujuan aplikasi ini:**
- Membantu mengelola order, menu, stok, supplier, dan pelanggan dengan lebih rapi.
- Ke depannya, aplikasi ini akan otomatis memesan ojek online untuk pengantaran.
- Ke depannya juga akan ada pengelolaan chat agar komunikasi dengan pelanggan lebih mudah.

Semoga aplikasi ini jadi saksi dari perjalanan sukses kalian.
